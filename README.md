# TechMove Logistics Application

## Overview

TechMove Logistics is a modern enterprise logistics management system developed using ASP.NET Core MVC.

The system is designed to streamline:
- Client Management
- Contract Management
- Service Request Processing
- Logistics Operations

The application features a modern responsive UI, validation, search/filtering, dashboard analytics, PDF uploads, and unit testing.

---

# Features

## Dashboard
- KPI cards
- Active/Expired contract monitoring
- Quick action buttons
- Responsive enterprise dashboard

## Client Management
- Create, Edit, Delete Clients
- Region dropdown selection
- Search & filtering
- Responsive UI

## Contract Management
- Contract lifecycle management
- Upload PDF agreements
- Service level tracking
- Contract status filtering
- Search functionality

## Service Request Management
- Create and manage service requests
- Currency conversion validation
- Cost tracking (USD & ZAR)
- Contract validation rules

## Validation Features
- PDF file validation
- Date validation
- Currency conversion validation
- Contract status validation
